## 운영 체제(operating system)

### 정의와 기능

운영 체제는 컴퓨터 하드웨어와 응용 프로그램을 제어 및 관리하는 **소프트웨어, 메인 프로그램**이다.  
그래서 사용자가 컴퓨터를 편리하고 효율적으로 사용할 수 있도록 도와준다.  
운영 체제가 없다면 하드웨어는 스스로 일할 수 없고 응용 프로그램도 실행될 수 없다.

- **대표적 하드웨어**  
  `CPU` 프로그램 실행을 위한 모든 **연산**을 처리  
  `메모리` CPU의 연산 결과를 **임시 저장**하고 프로그램 실행 공간을 제공  
  `디스크` 데이터와 프로그램을 **영구 저장**  
  `입출력, 네트워크 장치`

### 구조

운영 체제는 크게 **커널**(kernel)과 **쉘**(shell)로 구성되어 있다.

커널은 운영 체제의 핵심으로서 **하드웨어를 직접 제어**하며 운영 체제가 수행하는 모든 것을 저장한다. 운영 체제에서 하드웨어와 가장 가깝게 위치하여 소프트웨어의 요청을 하드웨어가 처리할 수 있도록 변환한다.

쉘은 명령어 해석기(command interpreter)라고도 불린다. 운영 체제에서 사용자와 가장 가깝게 위치하여 사용자의 **명령을 해석**하여 커널에 전달하고 **처리된 정보를 출력**한다. 명령은 GUI(Graphical User Interface)와 CLI(Command Line Interface) 같은 방식을 사용한다.

## 프로그램(program)

특정 작업을 수행하는 일련의 **명령어 모음**으로 실행 준비가 완료된 `정적 상태`의 파일이다.  
개발자가 프로그래밍 언어로 작성한 `코드`로 이루어져 있으며 보조기억 장치에 저장되어 있다.

- **응용 프로그램(application program)**  
  사용자가 실행하는 다양한 프로그램으로 `애플리케이션(application)`이라고도 한다.
